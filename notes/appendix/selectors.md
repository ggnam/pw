# CSS Selectors
```html
<p id="my-paragraph" class="red">This is a red paragraph.</p>
<input type="text" name="id" placeholder="아이디">
```

## Type Selectors
```css
p {
  color: red;
}
```

## Class Selectors
```css
.red {
  color: red;
}
```

## ID Selectors
```css
#my-paragraph {
  color: red;
}
```

## Attribute Selectors
```css
input[type="text"] {
  color: green;
}
```
